## 📋 Sobre o Projeto

O **ConsultMoney** é um painel web com 3 ferramentas financeiras desenvolvidas em JavaScript puro, criado como atividade prática da disciplina de Desenvolvimento em Javascript.

---

## 🛠️ Funcionalidades

### 1. 💱 Câmbio Comercial
- Converte valores em Reais para Dólar
- Aplica uma taxa de serviço de 3%
- Exibe o valor convertido, a taxa e o total final

### 2. 🛒 PDV - Aplicar Desconto
- Informa o preço de um produto
- Escolhe um cupom de desconto disponível:
  - `CLIENTE_NOVO` → 5% de desconto
  - `SOCIOTORCEDOR` → 10% de desconto
  - `BLACKFRIDAY` → 20% de desconto
- Exibe o preço original, o desconto e o preço final

### 3. 📈 Simulador de Investimento (CDB)
- Informa o capital inicial e a taxa de juros mensal
- Calcula o rendimento usando juros simples
- Exibe o capital, o rendimento e o total após 1 mês
